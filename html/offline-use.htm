Offline Use

 Top  Previous  Next

If you need to work offline on the shared project, or to make sure all sheets and issues are up to date,  download full cache from the Cloud/Shared location.

To download full cache:

1.In the main menu open Project > Open. Then Click on the Edit button in the toolbar.

2.Click the icon on the tile of the project you want to work with offline.

3.Choose the Load full cache option.  Revizto displays the task progress and then reverts to the project list (edit mode closed).
