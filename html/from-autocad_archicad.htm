AutoCAD

All in all, the overall export procedure is similar to the general one, but there are things to keep in mind.

AutoCAD allows working both with separate files and projects (sheet sets, .dst files); you can simultaneously have an open project and an open file outside this project. Note that in this case the export plug-in exports all open documents to Revizto. You have either to close redundant elements before starting export, or exclude them manually from the export.

Before exporting a 2D drawing from AutoCAD make sure to switch to the Paper space mode (the bottom menu of AutoCAD). Otherwise Revizto may fail to properly process exported data.

When defining your export options, make sure to click the Select button of the Export Sheets checkbox (see the image above) even if you are going to export all the sheets to check whether sheets are properly added to the export. Sometimes errors occur and sheets are not added to the export by default.
