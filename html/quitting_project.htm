Quitting a project

 Top  Previous  Next

You can quit a project team any time.

To quit a project team:

1.In the main menu open Project > Open. Then Click on the Edit button in the toolbar.

2.Click the icon on the tile of the project you want to work with offline.

3.Choose the Unsubscribe option and confirm your action.

Note that to get back on the project team you will have to re-invited by a project member with sufficient rights.
