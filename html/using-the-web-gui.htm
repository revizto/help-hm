Using the Web Workspace

 Top  Previous  Next

Each user (team member) has access to the web GUI where, depending on the access level, they can view or edit workspace and project settings, manage their personal profiles (account settings are available in Revizto application as well).

Note on regional settings: you cannot change your region. Any Revizto license is strictly linked to one of the specific geographies. You can change your language settings any time from the web GUI or in Revizto preferences.

Warning: Although the sign in page allows changing the region, you should log in from the region of your team license. Otherwise you will not be able to access your projects.
