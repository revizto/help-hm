Browsing Tips

When you browse the model, you can make all issue pins invisible: deactivate the icon at the bottom toolbar.

To quickly single out and select an object, use Ctrl+ Click on the object. Revizto highlights the selected object with a single color. You can select several objects, each will be highlighted in a different color. Object selection is handy for section cutting in the section box mode, as the option is immediately applied to you selection.
