Viewpoints

Initial viewpoints are exported from source files (mainly, from Revit). You can use them to navigate in the model. Also, you can create new viewpoints for better and faster navigation or to highlight an issue location.

You can select any viewpoint is the Home Viewpoint. (i.e. the one displayed when you click icon at the bottom of the screen.

Sample use of viewpoint

As mentioned above, you can create a viewpoint anywhere in the model. You can also create a viewpoint that uses custom color coding and sectioning settings. Then  you can create an issue in it and assign it to the author of the source file. They will see your viewpoint in Revizto while the source program will also automatically display the issue using your viewpoint settings.
