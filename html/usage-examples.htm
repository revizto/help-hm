Using 3D Controls to Create Issues

Example 1

To create an issue using section cut:

1.Launch your model and go to the 3D view (top menu option).

2.Walk to a place where the issue is located. You can go to a specific object, room etc. For example, let us navigate to a room (living room).

3.Then let us create a section ( icon) here. For example, let us cut the side wall of the room using the Plane mode of the section tool.

4.Then save the section as a viewpoint (the viewpoint icon at the bottom of the screen). This step is optional, if you just need to create an issue.

5.Now you can navigate to the newest viewpoint and create an issue containing the create section cut ( click ).

Example 2.

To create a section section box:

1.Walk/navigate to the spot where you have to create an issue.

2.Create a section using the Box mode. In the figure below you can see how a box containing a table is created.

3.Create a viewpoint with your section box (optional step).

4.Click to create an issue. Use markup tools to highlight key points.

Using 2D Controls to Create an Issue

Using 2D Ruler

1.Navigate to a sheet where you want to create an issue.

2.Use 2D ruler to take measurements of the spot where the issue is located.

3.Click the to create a new issue. Add more markup, if needed, and save you issue.
